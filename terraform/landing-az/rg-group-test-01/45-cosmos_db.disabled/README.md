# Azure Cosmos DB module

A basic example that uses the [Cosmos DB Terraform module](https://scm.capside.com/terraform/azure/terraform-azurerm-cosmos-db) to launch two databases.

## Requirements

| Name | Version |
|------|---------|
| <a name="requirement_terraform"></a> [terraform](#requirement\_terraform) | >=1.0.1 |
| <a name="requirement_azurerm"></a> [azurerm](#requirement\_azurerm) | >=2.61.0 |
| <a name="requirement_random"></a> [random](#requirement\_random) | 3.1.0 |
